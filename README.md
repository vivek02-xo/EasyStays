# EasyStay

Website Link: https://easystays-78u8.onrender.com/

EasyStay is a full-stack web application inspired by platforms like Airbnb, built to understand how real booking platforms work behind the scenes.

The project focuses on core features such as property listings, user authentication, reviews, and secure data handling — keeping the codebase clean, practical, and easy to extend.

---

## What EasyStay Does

- Users can create and manage property listings  
- View detailed listing pages with images and descriptions  
- Leave reviews and ratings  
- User authentication with login and signup  
- Basic authorization for protected actions  

---

## Tech Stack

- Frontend: HTML, CSS, Bootstrap  
- Backend: Node.js, Express  
- Database: MongoDB, Mongoose  
- Templating Engine: EJS (with layouts & partials)  
- Authentication: Passport.js  
- Validation: Joi  

---

## Purpose of This Project

EasyStay is built as a learning-driven project to:
- Practice full-stack development concepts  
- Understand RESTful routing and MVC structure  
- Work with authentication and authorization  
- Handle real-world validation and error handling  

This is not just a demo — it’s a foundation that can be scaled and improved.

---

Build features the right way.
